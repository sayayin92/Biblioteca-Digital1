## 🎯 Objetivo del proyecto
Desarrollar una aplicación backend en Go (Golang) que simula un sistema de biblioteca digital, permitiendo la gestión de usuarios, libros y préstamos.

El sistema aplica conceptos de estructuras (structs), manejo de datos en memoria y concurrencia mediante goroutines y mutex para simular el acceso simultáneo a recursos compartidos.

## ⚙️ Tecnologías utilizadas
- Go (Golang)
- goroutines, sync.Mutex, sync.WaitGroup
- fmt, bufio
- structs, slices, maps
- Git / GitHub

## 📋 Funcionalidades
- Registrar usuarios
- Registrar libros
- Listar libros
- Consultar disponibilidad
- Realizar préstamos de libros
- Validar disponibilidad
- Evitar doble préstamo
- Simulación de concurrencia con múltiples usuarios
